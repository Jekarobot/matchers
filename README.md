# 🎯 Matchers

Библиотека для сравнения объектов и значений с использованием гибких правил сопоставления (matchers). Позволяет писать читаемые и выразительные проверки в тестах и не только.

## 📖 Описание

Проект предоставляет набор функций-матчеров, которые упрощают написание проверок на соответствие данных ожидаемым шаблонам. Идеально подходит для unit-тестирования, валидации ответов API или фильтрации данных.

## 🛠 Технологический стек

- **Язык:** JavaScript (ES6+)
- **Среда выполнения:** Node.js
- **Тестирование:** Jest (предположительно, на основе структуры `coverage/`)
- **Сборка:** Babel (для транспиляции современного JS)

## 🚀 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/matchers.git
   cd matchers
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Запустите тесты:**
   ```bash
   npm test
   ```

4. **Соберите проект (если необходимо):**
   ```bash
   npm run build
   ```

## 💻 Примеры использования

```javascript
// Пример использования матчеров
const { match, isString, isNumber, isArray } = require('./src/js/matchers');

// Проверка, что значение является строкой
console.log(match('hello', isString())); // true

// Проверка, что значение является числом
console.log(match(42, isNumber())); // true

// Проверка массива
console.log(match([1, 2, 3], isArray())); // true
```
